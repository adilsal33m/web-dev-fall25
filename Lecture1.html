Course Objectives

Upon successful completion, the students will be able to

Understand the fundamental principles of web-based applications.
Gain practical experience with frontend and backend frameworks.
Design and implement secure and scalable web applications.
Work effectively in teams to deliver a complete deployed product.
Apply best practices in version control, testing, and deployment.

Course Outline (tentative)

Week 1

Topics: Course Introduction, Project Assignment, Source Control using Git, Wireframing

Outcome: Students form teams, receive project ideas, and learn to use version control. Draft project wireframes.

Week 2

Topics: HTML, CSS, Responsive Design

Outcome: Students can create landing page structure and implement responsive layout.

Week 3

Topics: CSS Frameworks, Javascript & DOM, Common UI Patterns

Outcome: Students can create styled pages with basic interactivity added.

Week 4

Topics: HTTP, Web Server, PHP, REST APIs

Outcome: Students can build forms and fetch data from backend.

Week 5

Topics: AJAX, jQuery and axios, Sessions & Cookies

Outcome: Students can send AJAX forms and update the page without full reload. Students can create session-aware elements.

Week 6

Topics: SSR vs SPA, Vue, Components, State Management, Routing

Outcome: Students gain awareness about two core ways to develop web applications. Students can create component-based pages.

Week 7

Topics: Laravel, Artisan, Database Design, Migration and Seeders

Outcome: Students can now create database schemas for storing data.

Week 8

Topics: ORM, Relations, Database Transactions, Collections

Outcome: Students can use ORM to access data.

Week 9

Topics: Middleware, Routing, Request Lifecycle, User Authentication

Outcome: Students can implement user authentication with secure password storage.

Week 10

Topics: Role Based Access Control, Caching, Queues

Outcome: Students can set up different roles and permissions. Students can cache information and run background tasks.

Week 11

Topics: API Design, Auth Token and JWT/Sanctum, Swagger/Postman, Testing

Outcome: Students can create API endpoints. Students can test their code.

Week 12

Topics: Notifications, Pagination, Accessibility, Reports

Outcome: Students can add features to improve user experience.

Week 13

Topics: Deployment, Containers, Logging, ML Backend (optional)

Outcome: Students can deploy applications, use containers, and handle logging.

Week 14

Topics: Web Security (XSS, CSRF, SQLi, MITM), HTTPS, Progressive Web Apps (optional)

Outcome: Students can identify and mitigate security risks in their application.

Week 15

Topics: Presentations

Outcome: Students present their final projects.
