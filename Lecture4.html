What is CSS?

  • CSS describes how HTML is presented: colors, layout, fonts, spacing.
  • “Cascading” = rules combine; conflicts are resolved by specificity and source order.
  • Styles live in stylesheets and target elements via selectors.
/* A tiny taste */

Three ways to specify CSS

1) External file (recommended)

<link rel="stylesheet" href="styles.css">

✔ Reusable, cacheable, keeps HTML clean.

2) <style> tag (page-specific)

<style>
  p { color: #374151; }
</style>

3) Inline (last resort)

<p style="color:tomato;">Hello</p>

⚠ Hard to maintain, highest specificity.

CSS Units

CSS units define lengths, sizes, and spacing. They can be absolute (fixed values) or relative (depend on parent, root, or viewport).

Absolute Units

Fixed sizes that do not change with screen size.

  • px – pixels (commonly used)
  • pt – points (print)
  • cm, mm, in – rarely used for screen
h1 { font-size: 32px; }
p { font-size: 12pt; }

Relative to Font Size

Scale relative to the font size of parent (em) or root (rem).

.parent { font-size: 16px; }
.child { font-size: 1.5em; } /* 24px */
.rem-text { font-size: 2rem; } /* relative to root */

Percentages

Percentages are relative to a parent property (like width or font-size).

.box { width: 50%; background: lightblue; }

Viewport Units

Relative to the browser window size.

  • vw – % of viewport width
  • vh – % of viewport height
  • vmin, vmax – smaller or larger of vw and vh

.full-height { height: 50vh; }

Boxes & the Box Model

Every element in CSS is a box. The box model describes how width, height, padding, border, and margin work together to determine the final size and spacing of an element.

Box Structure

The box model has four main layers:

  • Content – text or child elements inside
  • Padding – space between content and border
  • Border – line surrounding padding
  • Margin – space outside the element
.box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #3b82f6;
  margin: 20px;
}

Margin & Padding

Padding adds space inside the element; margin adds space outside.

.with-padding { padding: 1rem; }
.with-margin { margin: 1rem; }
.custom { margin: 1rem 2rem; /* top/bottom, left/right */ }

Responsive Design

Responsive design ensures that a website looks and works well on all screen sizes, from mobile to desktop.

Viewport

Always include the viewport meta tag to control scaling on mobile devices:

<meta name="viewport" content="width=device-width, initial-scale=1">

This allows the layout to respond to the actual screen width.

Fluid Layouts

Use relative units to create flexible layouts that adjust to screen width.

.container { width: 90%; max-width: 1200px; margin: 0 auto; }
.grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(150px, 1fr)); gap: 1rem; }
.flex { display: flex; flex-wrap: wrap; gap: 1rem; }

Media Queries

Media queries allow you to change styles based on screen size (breakpoints).

@media (max-width: 800px) {
  .header { grid-template-columns: 1fr; } /* stack columns */
  .container { padding: 1rem; }
}
@media (min-width: 1200px) {
  .container { max-width: 1400px; }
}

Responsive Spacing with CSS Variables

Use clamp() and variables to scale margins and padding fluidly.

.wrapper {
  max-width: 70ch;
  margin: 0 auto;
  padding: var(--space);
